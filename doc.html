Postup instalace

1. BACKEND

  • Nainstalujte PostgreSQL 17 (volitelně i pgAdmin pro GUI správu)
  • Zapamatujte si heslo, které nastavíte pro uživatele postgres
  • Ujistěte se, že nástroj createdb je dostupný v proměnné PATH (např. přidáním složky bin PostgreSQL do PATH)
  • Vytvořte databázi pisdb: 
    	createdb --username=postgres --encoding=UTF-8 pisdb
  • Nastavte připojení k databázi v EF Core pomocí dotnet user-secrets: 
    
	cd backend/API
	dotnet user-secrets init
	dotnet user-secrets set ConnectionStrings:PisDB "Host=localhost:5432; Username=postgres; Password=...; Database=pisdb"

    Poznámka: 'Password' je heslo uživatele postgres

  • Nainstalujte (nebo aktualizujte) EF Core CLI: 
    
	dotnet tool install --global dotnet-ef
  • Spusťte migrace pro inicializaci databázových tabulek: 
    
	cd backend/API
	dotnet ef database update --project ../DAL
  • Důvěřujte vývojovému HTTPS certifikátu: 
    
	dotnet dev-certs https --trust
  • Spusťte backend: 
    
	cd backend/API
	dotnet run
  • Server ponechte běžet a přejděte na část 2

2. FRONTEND

  • Ověřte, že máte nainstalovaný Node.js (doporučeno: verze 21.0.0 nebo novější) a npm
  • Přejděte do složky s frontendem: 
    
	cd frontend
  • Nainstalujte všechny závislosti uvedené v package.json: 
    
	npm install
  • Spusťte vývojový server: 
    
	npm run dev
  • Aplikace se zpravidla otevře na adrese http://localhost:5173 (Vite) nebo dle výstupu příkazu
